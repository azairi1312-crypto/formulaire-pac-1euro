# Formulaire PAC à 1€

Un site de landing page moderne pour la génération de leads sur les pompes à chaleur à 1€.

## Caractéristiques

- ✅ Formulaire complet avec 8 champs
- ✅ Design responsive (mobile & desktop)
- ✅ Dégradé violet/bleu moderne
- ✅ Section avantages avec 4 cartes
- ✅ Validation du formulaire côté client
- ✅ Message de succès/erreur

## Champs du formulaire

- Nom complet
- Email
- Téléphone
- Adresse (rue, code postal, ville)
- Type de chauffage actuel
- Surface du logement (m²)
- Type de logement

## Configuration GitHub Pages

1. Allez dans **Settings > Pages**
2. Sélectionnez **main** comme branche
3. Sélectionnez **/ (root)** comme dossier
4. Cliquez sur **Save**

Votre site sera accessible à : `https://azairi1312-crypto.github.io/formulaire-pac-1euro/`
